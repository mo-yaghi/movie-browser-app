# movie-browser-app

# How to Run the Project
Download or copy the project files to your computer
Open the project folder
Install dependencies:
npm install
Start the project:
npm start
Open your browser and go to:
http://localhost:3000

# Assumptions
All movies have valid data such as title, image, and rating.
The API always returns correct and complete results.
Users have a stable internet connection to fetch data.

# Known Limitations
No advanced filtering (e.g., by genre or year).
No user authentication or saved favorites feature.
Error handling is basic and may not cover all cases.
Some UI elements are simplified for clarity.
